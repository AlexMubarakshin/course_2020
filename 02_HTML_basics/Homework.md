# Домашнее задание второго занятия

## Первое задание

На основе изученных тегов, доработайте задание с прошлого урока.

На странице должны быть использованны следующие теги:

- `<div>`
- `<b>`
- `<main>`
- `<p>`
- `<ul>`

Так же в разделе `<head>`, добавить:

- `<title>`
- `<meta name="description" ... >`
