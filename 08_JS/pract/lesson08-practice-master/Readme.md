# Блокнот

Простой блокнот

## Запуск

```bash
# Установить библиотеки
$ npm

# Запуск webpack сервера и линтеров
$ npm start
```

## Доступные команды

```bash
# Запуск всех линтеров
$ npm run validate-all

# Запуск CSS линтера
$ npm run validate-css

# Запуск HTML линтера
$ npm run validate-html

# Запуск JS линтера
$ npm run validate-js

# Запуск webpack в режиме watch, без dev-сервера
$ npm run watch
```
